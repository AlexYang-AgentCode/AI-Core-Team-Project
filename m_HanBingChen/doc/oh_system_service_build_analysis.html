1. еҲҶжһҗиғҢжҷҜдёҺзӣ®ж Ү

1.1 еҲҶжһҗиғҢжҷҜ

Android-OH йҖӮй…ҚйЎ№зӣ®зҡ„ж ёеҝғжһ¶жһ„жҳҜпјҡеңЁ OH зі»з»ҹдёҠиҝҗиЎҢ Android AppгҖӮAndroid App иҝӣзЁӢеҶ…дҝқз•ҷ AOSP Framework е®ўжҲ·з«Ҝе’Ң ART иҷҡжӢҹжңәпјҢзі»з»ҹжңҚеҠЎдҫ§еҲҷе®Ңе…ЁдҪҝз”Ё OH еҺҹз”ҹзі»з»ҹжңҚеҠЎгҖӮйҖӮй…ҚеұӮпјҲAdapterпјүдҪңдёәжЎҘжўҒпјҢжӢҰжҲӘ Android App еҸ‘еҮәзҡ„ Binder IPC и°ғз”ЁпјҢиҪ¬жҚўдёә OH зі»з»ҹжңҚеҠЎзҡ„ IPC иҜ·жұӮгҖӮ

еүҚдёҖд»ҪжҠҘе‘ҠпјҲгҖҠAOSP 14 жңҖе°ҸеҢ–жһ„е»әеҲҶжһҗжҠҘе‘ҠгҖӢпјүеҲҶжһҗдәҶ Android дҫ§"App иҝҗиЎҢж—¶"зҡ„зј–иҜ‘иЈҒеүӘж–№жЎҲгҖӮжң¬жҠҘе‘Ҡд»Һ OH дҫ§еҮәеҸ‘пјҢеҲҶжһҗйҖӮй…ҚеұӮжүҖйңҖзҡ„ OH зі»з»ҹжңҚеҠЎжЁЎеқ—иҜҘеҰӮдҪ•зј–иҜ‘жһ„е»әгҖӮ

1.2 дёҺ AOSP жһ„е»әзҡ„еҜ№з§°е…ізі»

з»ҙеәҰAOSP дҫ§пјҲе·ІеҲҶжһҗпјүOH дҫ§пјҲжң¬жҠҘе‘Ҡпјү
зј–иҜ‘еҶ…е®№ Framework е®ўжҲ·з«Ҝд»Јз ҒпјҲframework.jarпјү
ART иҷҡжӢҹжңә / C иҝҗиЎҢж—¶		 зі»з»ҹжңҚеҠЎз«Ҝд»Јз ҒпјҲеҗ„ SA зҡ„ .z.soпјү
IPC е®ўжҲ·з«Ҝеә“пјҲinner_kitsпјү
жҺ’йҷӨеҶ…е®№ system_server еҸҠжүҖжңүзі»з»ҹжңҚеҠЎ OH зҡ„ App жЎҶжһ¶еұӮпјҲArkUIгҖҒNAPI зӯүпјү
жһ„е»әзі»з»ҹ Soong (Android.bp / Blueprint) GN + Ninja (BUILD.gn)
дә§зү©з”ЁйҖ” App иҝӣзЁӢеҶ…иҝҗиЎҢ йҖӮй…ҚеұӮ JNI еә“й“ҫжҺҘ + OH зі»з»ҹжңҚеҠЎиҝҗиЎҢ
жәҗз Ғдҝ®ж”№ 4 дёӘ AOSP ж–Үд»¶пјҲaosp_patches/пјү 2 дёӘ OH жЁЎеқ—зҡ„еӨҡдёӘж–Үд»¶пјҲohos_patches/пјү
е…ій”®еҢәеҲ«пјҡAOSP дҫ§еҸӘйңҖзј–иҜ‘е®ўжҲ·з«ҜпјҲжҺ’йҷӨжңҚеҠЎз«ҜпјүпјҢеӣ жӯӨжҳҜ"иЈҒеүӘ"гҖӮOH дҫ§йңҖиҰҒе®Ңж•ҙзҡ„зі»з»ҹжңҚеҠЎеңЁи®ҫеӨҮдёҠиҝҗиЎҢпјҢеҗҢж—¶йҖӮй…ҚеұӮиҝҳйңҖиҰҒй“ҫжҺҘ OH зҡ„ IPC е®ўжҲ·з«Ҝеә“пјҲinner_kitsпјүпјҢеӣ жӯӨжҳҜ"жҢүйңҖйҖүжӢ© + иЎҘдёҒеўһејә"гҖӮ

1.3 жһ„е»әзӣ®ж Ү

йңҖиҰҒд»Һ OH зј–иҜ‘иҺ·еҸ–зҡ„еҶ…е®№

  • OH зі»з»ҹжңҚеҠЎ .z.so вҖ” AbilityManagerServiceгҖҒWindowManagerServiceгҖҒBundleManagerService зӯүзі»з»ҹ SA зҡ„е…ұдә«еә“пјҢжү“иЎҘдёҒеҗҺйҮҚж–°зј–иҜ‘йғЁзҪІеҲ°и®ҫеӨҮ
  • Inner Kits е®ўжҲ·з«Ҝеә“ вҖ” йҖӮй…ҚеұӮ JNI д»Јз Ғй“ҫжҺҘ OH жңҚеҠЎж—¶йңҖиҰҒзҡ„еӨҙж–Үд»¶е’Ңе®ўжҲ·з«Ҝеә“пјҲеҰӮ ability_managerгҖҒapp_managerгҖҒlibwmгҖҒlibrender_service_client зӯүпјү
  • йҖӮй…ҚеұӮиҮӘжңүжЁЎеқ— вҖ” appspawn-xпјҲж··еҗҲеӯөеҢ–еҷЁпјүгҖҒapk_installerпјҲAPK е®үиЈ…еә“пјүгҖҒoh_adapter_bridgeпјҲJNI жЎҘжҺҘеә“пјүпјҢйҖҡиҝҮ OH жһ„е»әзі»з»ҹзј–иҜ‘дёә OH дёҠиҝҗиЎҢзҡ„дәҢиҝӣеҲ¶

дёҚйңҖиҰҒд»Һ OH зј–иҜ‘иҺ·еҸ–зҡ„еҶ…е®№

  • ArkUI / ArkTS / NAPI вҖ” Android App дёҚдҪҝз”Ё OH зҡ„ UI жЎҶжһ¶
  • OH еә”з”ЁжЎҶжһ¶пјҲFA/Stage жЁЎеһӢзҡ„ App дҫ§д»Јз Ғпјү вҖ” Android App жңүиҮӘе·ұзҡ„ Framework
  • OH Launcher / SystemUI / Settings вҖ” и®ҫеӨҮдёҠе·ІжңүпјҢж— йңҖйҖӮй…ҚеұӮеҸӮдёҺ
  • HAL / HDI еұӮ вҖ” з”ұеҺӮе•ҶжҸҗдҫӣпјҢдёҚеңЁжң¬йЎ№зӣ®зј–иҜ‘иҢғеӣҙ

дәӨеҸүзј–иҜ‘пјҡж ёеҝғжһ¶жһ„й—®йўҳ

е…ій”®зәҰжқҹпјҡAOSP зј–иҜ‘дә§зү©пјҲlibart.so зӯүпјүй“ҫжҺҘзҡ„жҳҜ bionic libcпјҢиҖҢ OH зі»з»ҹдҪҝз”Ё musl libcгҖӮдёӨз§Қ C иҝҗиЎҢж—¶зҡ„ TLSгҖҒpthreadгҖҒmalloc е®һзҺ°е®Ңе…ЁдёҚеҗҢпјҢж— жі•зӣҙжҺҘе°Ҷ bionic зј–иҜ‘зҡ„ .so еӨҚеҲ¶еҲ° OH дёҠдҪҝз”ЁгҖӮеҝ…йЎ»з”Ё OH е·Ҙе…·й“ҫпјҲ--target=aarch64-linux-ohosпјүйҮҚж–°зј–иҜ‘ AOSP C++ жәҗз ҒпјҢдҪҝдә§зү©й“ҫжҺҘ musl libcгҖӮ

4 еұӮдәӨеҸүзј–иҜ‘зӯ–з•Ҙ

з”ұдәҺ AOSP зҡ„ C++ жәҗз Ғдҫқиө– Bionic дё“жңү APIпјҲзі»з»ҹеұһжҖ§гҖҒmalloc жү©еұ•гҖҒfdsan зӯүпјүпјҢйңҖиҰҒе…ҲеҲӣе»әдёҖдёӘ Bionic е…је®№еұӮпјҢз„¶еҗҺеҲҶеұӮзј–иҜ‘пјҡ

з¬¬ 1 еұӮпјҡlibbionic_compat.so пјҲOH жһ„е»әпјү в”ңв”Җв”Җ Bionic дё“жңү API вҶ’ OH жӣҝд»Је®һзҺ° в”Ӯ в”ңв”Җв”Җ __system_property_set() вҶ’ OH SystemSetParameter() в”Ӯ в”ңв”Җв”Җ android_mallopt() вҶ’ з©әе®һзҺ° в”Ӯ в””в”Җв”Җ android_fdsan_*() вҶ’ з©әе®һзҺ° в”ңв”Җв”Җ дҫқиө–пјҡOH musl libc + init:libbegetutil в””в”Җв”Җ дә§зү©пјҡlibbionic_compat.so + е…је®№еӨҙж–Үд»¶ з¬¬ 2 еұӮпјҡAndroid еҹәзЎҖеә“ пјҲз”Ё OH Clang йҮҚзј– AOSP жәҗз Ғпјү в”ңв”Җв”Җ liblog.so, libcutils.so, libutils.so, libbase.so, libnativehelper.so в”ңв”Җв”Җ дҫқиө–пјҡз¬¬ 1 еұӮ + musl libc в””в”Җв”Җ зј–иҜ‘пјҡOH Clang --target=aarch64-linux-ohos з¬¬ 3 еұӮпјҡART иҷҡжӢҹжңә пјҲз”Ё OH Clang йҮҚзј– AOSP art/ жәҗз Ғпјү в”ңв”Җв”Җ libart.so, libart-compiler.so, libdexfile.so, libsigchain.so в”ңв”Җв”Җ дҫқиө–пјҡз¬¬ 1-2 еұӮ в””в”Җв”Җ е…ій”®пјҡART е·Іжңү #if !defined(__BIONIC__) fallback и·Ҝеҫ„ з¬¬ 4 еұӮпјҡlibandroid_runtime.so пјҲиЈҒеүӘзүҲпјҢз”Ё OH Clang йҮҚзј–пјү в”ңв”Җв”Җ дҝқз•ҷ ~25 дёӘ JNI жЁЎеқ—пјҲеҺ»жҺү HAL/и“қзүҷ/NFC/зі»з»ҹжңҚеҠЎ JNIпјү в”ңв”Җв”Җ дҫқиө–пјҡз¬¬ 1-3 еұӮ + еӣҫеҪўеә“ в””в”Җв”Җ дә§зү©пјҡlibandroid_runtime.soпјҲиЈҒеүӘзүҲпјү
еҸҜиЎҢжҖ§дҫқжҚ®пјҡART жәҗз Ғдёӯе·ІжңүеӨ§йҮҸ #if defined(__BIONIC__) ... #else ... жқЎд»¶зј–иҜ‘еҲҶж”ҜпјҢеӣ дёә ART ж”ҜжҢҒеңЁ Linux host дёҠзј–иҜ‘иҝҗиЎҢпјҲз”ЁдәҺ dex2oat зӯүе·Ҙе…·пјүгҖӮдәӨеҸүзј–иҜ‘ж—¶ __BIONIC__ жңӘе®ҡд№үпјҢиҮӘеҠЁиө°йҖҡз”Ёи·Ҝеҫ„пјҲеҰӮ pthread_setspecific жӣҝд»Ј bionic TLS slotпјүгҖӮ
