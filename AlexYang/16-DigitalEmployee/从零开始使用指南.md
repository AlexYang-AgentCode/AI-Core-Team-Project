# 🚀 从零开始使用 OpenCode + tmux

> 一步步教你如何进入和使用

---

## 📋 第一步：进入 WSL

### 方法1: 从 Windows 开始菜单
```
1. 点击 Windows 开始按钮
2. 搜索 "Ubuntu" 或 "WSL"
3. 点击打开
```

### 方法2: 从 PowerShell/CMD
```powershell
# 输入命令
wsl
```

### 方法3: 从 Windows Terminal
```
1. 打开 Windows Terminal
2. 点击下拉箭头 ↓
3. 选择 "Ubuntu" 或你的 WSL 发行版
```

---

## 🎯 第二步：启动 OpenCode (两种方式)

### 方式A: 使用 tmux（推荐）

#### 1️⃣ 启动 tmux 会话

```bash
# 方法1: 使用快捷脚本（最简单）
~/bin/tmux-opencode

# 方法2: 手动创建
tmux new -s opencode
cd /mnt/d/ObsidianVault/10-Projects/16-DigitalEmployee
oc
```

#### 2️⃣ 进入 OpenCode

脚本会自动启动 OpenCode，你会看到：
```
🚀 创建新的 OpenCode 会话...
✅ 会话已创建: opencode

📌 tmux 使用提示:
   • 鼠标滚轮: 直接滚动查看历史
   • 退出会话: Ctrl+a 然后按 d
   • 恢复会话: tmux attach -t opencode
   • 分屏: Ctrl+a 然后按 | 或 -

[OpenCode 界面出现]
```


## 🎮 tmux 核心操作指南

### 启动 OpenCode 后的操作

| 你想做什么 | 操作步骤 | 说明 |
|-----------|---------|------|
| **查看历史输出** | 直接用鼠标滚轮 | ⬆️⬇️ 自由滚动 |
| **搜索输出** | `Ctrl+a [` 进入复制模式，然后按 `/` | 输入关键词搜索 |
| **暂时退出** | `Ctrl+a` 然后按 `d` | 会话在后台运行 |
| **恢复会话** | `tmux attach -t opencode` | 回到之前的工作 |
| **分屏显示** | `Ctrl+a \|` 或 `Ctrl+a -` | 垂直或水平分屏 |
| **切换分屏** | `Ctrl+a` + 方向键 | 在分屏间切换 |

### 详细操作示例

#### 示例1: 查看历史输出（最重要）

```
情况: OpenCode 输出了很长的内容，你想向上查看

操作:
1. 直接用鼠标滚轮 ⬆️ 向上滚动
   或
2. 按 Ctrl+a [ 进入复制模式
3. 用 ↑↓ 或 PgUp/PgDn 滚动
4. 按 q 退出复制模式
```

#### 示例2: 暂时离开去喝咖啡

```
操作:
1. 按 Ctrl+a（先按Ctrl，再按a）
2. 松开后按 d（detach）
3. 你会退出到普通终端，但 OpenCode 还在后台运行

回来后:
tmux attach -t opencode
# 又回到刚才的状态，所有输出都还在！
```

#### 示例3: 分屏查看日志

```
操作:
1. 按 Ctrl+a |（垂直分屏）
   现在有两个窗口
   
2. 在右边窗口输入:
   tail -f ~/opencode-logs/*.log
   
3. 按 Ctrl+a ← 切换回左边继续使用 OpenCode
```

## 💡 快速参考卡

### tmux 快捷键速记

```
Ctrl+a ?        显示所有快捷键
Ctrl+a d        退出会话（后台运行）
Ctrl+a [        进入复制模式（可滚动、搜索）
Ctrl+a ]        粘贴

Ctrl+a |        垂直分屏
Ctrl+a -        水平分屏
Ctrl+a 方向键    切换分屏

Ctrl+a c        创建新窗口
Ctrl+a 数字      切换到第N个窗口
Ctrl+a w        列出所有窗口
```

### OpenCode 快捷命令

```bash
oc              # 启动 OpenCode（自动日志）
ocl             # 查看最新日志
opencode-logs stats    # 查看统计
opencode-logs view     # 查看最新日志
opencode-logs search X # 搜索日志
```

## 🚨 常见问题

### Q1: 我按了 Ctrl+a d，怎么回来？

```bash
# 查看所有会话
tmux ls

# 恢复会话
tmux attach -t opencode
# 或简写
tmux a -t opencode
```

### Q2: tmux 里鼠标滚动不工作？

```bash
# 确认配置已加载
tmux source-file ~/.tmux.conf

# 或者在 tmux 中按 Ctrl+a 重载
Ctrl+a r
```

### Q3: 如何完全退出 tmux？

```bash
# 方法1: 在 tmux 会话中
exit

# 方法2: 从外部杀死会话
tmux kill-session -t opencode
```

### Q4: 找不到 ~/bin/tmux-opencode？

```bash
# 手动创建
mkdir -p ~/bin

cat > ~/bin/tmux-opencode << 'EOF'
#!/bin/bash
SESSION_NAME="opencode"
PROJECT_ROOT="/mnt/d/ObsidianVault/10-Projects/16-DigitalEmployee"

tmux has-session -t $SESSION_NAME 2>/dev/null

if [ $? != 0 ]; then
    echo "🚀 创建新的 OpenCode 会话..."
    tmux new-session -d -s $SESSION_NAME -c $PROJECT_ROOT
    tmux send-keys -t $SESSION_NAME "oc" C-m
else
    echo "✅ 会话已存在: $SESSION_NAME"
fi

tmux attach -t $SESSION_NAME
EOF

chmod +x ~/bin/tmux-opencode
```
