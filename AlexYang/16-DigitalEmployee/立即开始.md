## 🚀 立即使用 (3步)

### 第1步: 在 VSCode 中打开项目

```bash
# 在终端运行
code /mnt/d/ObsidianVault
```

### 第2步: 打开集成终端

```
按键: Ctrl + ` (反引号)
```

### 第3步: 启动 OpenCode

```bash
# 重载配置（第一次需要）
source ~/.bashrc

# 进入项目
cd 10-Projects/16-DigitalEmployee

# 启动 OpenCode (自动记录日志)
oc
```

## 📋 快捷命令速查

```bash
# OpenCode 相关
oc              # OpenCode + 日志 (推荐)
ocd             # OpenCode + DeepSeek
ock             # OpenCode + Kimi
ocg             # OpenCode + GLM

# 日志管理
ocl             # 查看最新日志
opencode-logs stats    # 查看统计

opencode-logs search X # 搜索

# 其他
proj            # 快速进入项目目录
token-stats     # Token 使用统计
```
